- [App Type](#app_type)
    - The app type is now set using the `NEXT_PUBLIC_APP_TYPE` variable. Setting this to personal will turn the rich snippets schema data into a personal website.

- [Config Namespace](#config_namespace)
    - Created a 'config/' namespace to set a base for all environment variables.


- [Core Namespace](#core_namespace)
    - Created a 'core/' namespace within the components and utility directories. This will serve as the directories for core Elegant code, and will make future upgrades a breeze.

- [Environment File](#environment_file)
    - Added slimmed down starter .env file during project creation that contains only the basic variables to get started with a new app.
    - Added a complete .env.example file that contains all variables.
    - Added a .env.test file for complete Jest unit testings with variables.


- [Favicons](#favicons)
    - Updated the favicons to be proper across all browsers.


- [Meta.js Component](#meta_js_component)
    - Deprecated the Meta.js component; Will be removed by version 3. The Meta component is replaced by our Next-Seo implementation


- [Seo Component](#seo_component)
    - Added complete core SEO components that user would expect, but are "plugins" within WordPress.
    - Added Seo Rich snippets, including the brands logo.
